Skiller 是一款为中文用户打造的 Claude Skills 浏览器，汇集 1400+ 高质量 AI 技能，从编程、写作到数据分析，一键找到适合的 Claude Skill。

## 核心功能

- 海量技能库：Anthropic 官方 + 社区精选，每日自动同步
- 中文详解：每个 Skill 配有中文简介、使用场景与操作指引
- 智能搜索：按关键词、分类快速定位
- 收藏管理：一键收藏常用 Skill

## 技术栈

- React Native + Expo (SDK 54)
- Supabase (PostgreSQL)
- Node.js 数据管道 + DeepSeek 中文内容生成
- GitHub Actions 每日自动同步
